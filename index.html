Nodos

Todo elemento HTML es un nodo, pero no todo nodo es un elemento HTML

  • Nodo de elemento (p o a los selectores)
  • Nodo de atributo (clase o id)
  • Nodo de Texto (contenido)

TextConent

El textContent permite ver que texto tiene el elemento, a la vez se puede utilizar para cambiarlo o aplicarlo a un elemento nuevo.

message.textContent = 'Soy un mensaje cambiado desde Javascript';

Estilos

Se pueden aplicar estilos de CSS dentro de JavaScript, para esto es necesario el tratar con un elemento individual o transformar una lista de nodos en items individuales

const message = document.getElementById('message'); message.style.color = 'red';

const linkarray = Array.from(document.querySelectorAll('.NavBar li a'));

for (i=0;i>=linkarray.length-1;i++) { linkarray[i].style.color = 'red'; }

createElement

container.appendChild(divElement);
